# **App Name**: Sui at home

## Core Features:

- Recipe Browsing & Detail View: Public users can view a homepage with featured/latest/popular recipes, browse all recipes, filter by category, search, and access detailed recipe pages with cover image, title, description, story, ingredients, steps, tips, and counts for likes, favorites, and comments.
- User Authentication & Profiles: Public visitors can sign up and log in. Logged-in users have a profile page to manage saved/favorited recipes.
- Interactive Recipe Actions: Users can 'like' recipes (public), and logged-in users can 'save', 'favorite', and 'comment' on recipes. Like counts are displayed on recipe cards and detail pages.
- Admin Recipe Management: A secure admin dashboard allows the creator to create, edit, and delete recipe posts, upload cover photos, add all recipe details (title, description, ingredients, steps, etc.), and manage user comments.
- Basic Recipe Analytics Dashboard: The admin can view key metrics such as total users, recipes, likes, favorites, comments, and views. Simple graphs for most liked, favorited, commented, and viewed recipes, along with popular categories.
- AI Recipe Enhancer Tool: Admin tool that uses generative AI to suggest engaging 'story or intro about the dish' content and creative 'cooking tips' based on recipe ingredients or title inputs, helping to enrich recipe descriptions.
- Mobile-First Responsive Design: The entire application, including all pages and content, is fully responsive and optimized for seamless viewing and interaction across various mobile devices and screen sizes, particularly catering to Instagram traffic.

## Style Guidelines:

- Primary background: Peach (#F0B593) for a warm and welcoming base.
- Secondary background/accents: Soft Green (#B4E194) for natural elements and Dusty Blue (#89C4CE) for a clean, calming feel in UI elements.
- Accent color: Warm Yellow (#F2C258) for interactive elements, highlights, and calls-to-action to stand out.
- Text and strong contrasts: Charcoal (#313131) for excellent readability.
- Headline font: 'Playfair' (serif), with its elegant, high-contrast design, to convey a high-quality, artful feel for recipe titles and primary headings.
- Body font: 'PT Sans' (humanist sans-serif), for its modern yet warm character and excellent readability in longer texts such as recipe stories, ingredients, and steps. This pairing ensures both elegance and legibility across all content.
- Clean, delicate line-art icons used consistently throughout the application, reflecting culinary themes without being overly cartoonish. Icons for actions like like, favorite, save, and comment will be immediately recognizable and contribute to the app's modern and clean aesthetic.
- A food-first, grid-based layout emphasizing high-quality recipe imagery. Ample whitespace ensures content clarity and a sophisticated feel. Responsive design prioritized, with thoughtful adaptations for mobile to ensure a comfortable and intuitive experience for users accessing from Instagram.
- Subtle and delightful micro-interactions. Hover effects on recipe cards, gentle transitions for loading new content, and a satisfying, brief animation when users 'like' or 'save' a recipe. Animations will be minimal, fluid, and contribute to the warm and polished user experience.
